Getting Started with Inatech Design System

This guide explains how to setup your Angular project to begin using Design System. We can integrate Design system in two methods.

  1. Package method
  2. Manual method

Package method

In this method, we use the Angular CLI's install schematic to set up the Design system project by running the following command:

npm install inatech-design-system@5.0.0

You can also check the previous version of design system.

Manual method

To manually add the Design System into an application, copy required folders from GitHub URL or Zip file.

  • To add all shared components, copy the folder path src/app/shared
  • To add global css styles, copy the folder path src/themes/defaultweb
  • To add custom icons and images, copy the folder path src/assets/customicons and src/assets/customimages

How to do packaging in Design System?

  • Fetch the latest source code from the repository.
  • Update package version in ‘package.json’ file.
  • Update package version in ‘readme.md’ file.
  • Update package version in ‘designsystem.component.ts’ file.
  • Delete ‘dist’ folder.
  • To publish the package, use the command ‘npm run vsts-publish’.
  • Open artifacts in vsts and verify the package version.
  • Once successfully published, commit the latest version file changes into Git.
